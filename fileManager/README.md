# fileManager

Десктопное приложение для управления файлами, разработанное на **.NET 10** с использованием кроссплатформенного UI-фреймворка **Avalonia UI**.

## 📋 Возможности

- Просмотр содержимого директорий (файлы и папки)
- Создание новых файлов
- Создание новых папок
- Удаление файлов и папок
- Открытие и просмотр содержимого текстовых файлов
- Сохранение изменений в файлах
- Обновление списка файлов

## 🛠 Технологии

- **.NET 10** — платформа разработки
- **Avalonia UI 11.3.11** — кроссплатформенный UI-фреймворк
- **CommunityToolkit.Mvvm 8.2.1** — библиотека для реализации паттерна MVVM
- **Fluent Theme** — современный дизайн интерфейса

## 📖 Использование

При запуске приложение отображает содержимое **рабочего стола**.

### Кнопки управления:

| Кнопка | Описание |
|--------|----------|
| **Создать файл** | Создаёт новый файл `NewFile.txt` в текущей директории |
| **Создать папку** | Создаёт новую папку `NewFolder` в текущей директории |
| **Удалить** | Удаляет выбранный файл или папку |
| **Открыть** | Открывает содержимое выбранного файла для просмотра/редактирования |
| **Сохранить** | Сохраняет изменения в выбранном файле |
| **Обновить** | Обновляет список файлов в текущей директории |

### Модель данных `FileItem`:

```csharp
public class FileItem
{
    public string Name { get; set; }      // Имя файла/папки
    public string FullPath { get; set; }  // Полный путь
    public long Size { get; set; }        // Размер в байтах
    public DateTime Created { get; set; } // Дата создания
    public bool IsDirectory { get; set; } // true для папки, false для файла
}
```

## 🏗 Архитектура

Приложение построено по паттерну **MVVM** (Model-View-ViewModel):

- **Model** (`FileItem`) — представляет данные о файлах и папках
- **View** (`MainWindow`) — UI-слой на Avalonia UI с XAML-разметкой
- **ViewModel** (`MainWindowViewModel`) — содержит бизнес-логику и команды

### Основные компоненты:

- **ObservableProperty** — атрибут для автоматической генерации свойств с уведомлением об изменениях
- **RelayCommand** — атрибут для создания команд, привязанных к кнопкам UI
- **ViewLocator** — автоматически связывает ViewModel с соответствующим View
